Human-facing documentation for the Swift Institute ecosystem.

This catalog is for readers evaluating, using, or writing about the Swift Institute ecosystem. For development conventions applied while writing code inside the ecosystem, see the `Skills/` directory — that is the canonical source for naming, errors, memory safety, testing, and related rules.

For the repository overview and the ecosystem map, see the [repository README](../README.md). For the organization-level view across all Swift Institute orgs, see the [organization profile](https://github.com/swift-institute).
