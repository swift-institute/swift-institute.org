## Next Steps

Retest all experiments on next toolchain upgrade. Update status to CONFIRMED if fixed, or keep BUG REPRODUCED if still present.

Priority order for revalidation:
1. **@_rawLayout deinit chain** (3 storage-primitives experiments + noncopyable-nested-deinit-chain) — blocks removal of `_deinitWorkaround` from 21 types
2. **~Copyable constraint poisoning** (5 experiments) — blocks direct Sequence conformance for ~Copyable containers
3. **DiagnoseStaticExclusivity crash** (exclusivity-crash-small-buffer) — blocks ergonomic optional chaining on ~Copyable accessor chains
4. **unsafe for-in release crash** — blocks expression-level `unsafe` adoption
5. **CopyPropagation #87029** (phantom-tagged-string-unification V6) — blocks release-mode phantom-tagged string without @_optimize(none)
6. **Remaining language limitations** (throws overloading, protocol coroutine accessors, witness matching) — may require Swift Evolution proposals
