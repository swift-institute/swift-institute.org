## [0.1.0] — Initial public alpha

Documentation, conventions, research, experiments, and blog workflow made
public. The Swift package layers described here (Primitives, Standards,
Foundations) are being released repository by repository over the
following weeks.

- `README.md`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`,
  `LICENSE.md`, and `.github/profile/README.md` set up the public
  OSS baseline.
- `Documentation.docc/` documents the five-layer architecture, shared
  vocabulary, and FAQ.
- `Skills/` hosts the canonical conventions (naming, errors, memory safety,
  testing, modularization, platform, Swift Evolution process, research
  and blog workflows).
- `Research/` and `Experiments/` carry design rationale and runnable
  receipts respectively.
- `Blog/` contains drafts staged for publication under the receipts model.
- `Swift Evolution/` tracks draft pitches and proposal phase routing.
