# Blog Post Process

Workflows for creating and publishing blog posts.

---

## Stages

### 1. Ideation
- Capture ideas in `Blog/_index.md`
- Include source reference (experiment, research, discovery)
- Tag with potential category

### 2. Drafting
- Create draft in `Blog/Draft/`
- Follow style guide in `Blog/_Styleguide.md`
- Include code examples

### 3. Review
- Move to `Blog/Review/`
- Technical accuracy check
- Style consistency check

### 4. Publishing
- Final polish
- Move to published location

---

## Blog Post Template

```markdown
# [Title]

## Summary
One paragraph overview.

## The Problem
What challenge are we addressing?

## The Solution
Technical explanation with code.

## Key Takeaways
Bullet points for quick reference.

## References
Related resources.
```
